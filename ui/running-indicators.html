“What’s running?” — indicator options

Eight ways to mark the live node — OpenAI-themed: monochrome green, separated from “done” by motion, not a second hue (the way ChatGPT marks thinking/streaming). Pills note where each works best — heavy motion shines for one live node but turns to noise when several run at once. The bottom row shows the key contrast: running animates, done is static.
